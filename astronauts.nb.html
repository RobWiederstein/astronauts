Astronauts

This project was based on the astronaut dataset at kaggle. It was downloaded and imported. There were 1277 rows and 24 columns.

input <- "./data/astronauts.csv"
df <- data.table::fread(input = input)

Missingness

df.na <- df[!complete.cases(df), ]
